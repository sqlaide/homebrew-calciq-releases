# homebrew-calciq-releases

Homebrew tap for [calciq](https://www.simplemind.dev/calciq) — a terminal-native calculator.

## Install

```bash
brew tap sqlaide/calciq-releases
brew install calciq
```

## Usage

```bash
calciq                  # interactive TUI
calciq "2500 * 1.2"    # one-shot
calciq --help
```
